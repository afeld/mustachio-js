---

## Requirements:

* NodeJS v0.4.1+
* ImageMagick v6
* caching (recommended)

## Installation

* Set `MUSTACHIO_FACE_API_KEY` and `MUSTACHIO_FACE_API_SECRET` environment variables - see [developer.face.com](http://developers.face.com).
* Set `PORT` - otherwise defaults to 8080
